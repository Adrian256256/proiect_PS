# GSM Signal Monitor - Python Dependencies

# All dependencies are part of Python standard library
# Python 3.7+ required

# System Dependencies (install separately):
# macOS:
#   brew install rtl-sdr kalibrate-rtl
# Linux (Debian/Ubuntu):
#   sudo apt-get install rtl-sdr kalibrate-rtl librtlsdr-dev
# Windows:
#   Download from https://www.rtl-sdr.com/

# Standard library modules used:
# - tkinter (GUI framework)
# - subprocess (RTL-SDR command execution)
# - threading (background scanning)
# - re (regex parsing)
# - time, datetime (timing and timestamps)
# - dataclasses (data structures)

# External dependencies:
matplotlib>=3.5.0
# - typing (type hints)
# - os, tempfile (file operations)

# No pip packages required for basic functionality
